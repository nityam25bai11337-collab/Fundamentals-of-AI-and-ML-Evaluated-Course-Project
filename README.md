# Student Performance Predictor

## Project Overview
This project is a simple Machine Learning application that predicts a student's marks based on two important factors:
- Hours studied
- Attendance percentage

The idea behind this project is to understand how student effort and attendance affect academic performance.

---

-->Why This Project?
In real life, teachers often try to estimate student performance based on behavior like study time and attendance.  
This project uses Machine Learning to automate that prediction.

It helps answer questions like:
- If a student studies more, how much will marks improve?
- Does attendance really impact performance?

---

--> How the Project Works

The project follows a basic Machine Learning workflow:

1. -->Data Preparation**
   - A dataset is created with study hours, attendance, and marks.

2. -->Feature Selection**
   - Inputs: Hours studied, Attendance  
   - Output: Marks

3. -->Train-Test Split**
   - Data is divided into training and testing sets.

4. -->Model Training**
   - A Linear Regression model is trained using the data.

5. -->Prediction**
   - The model predicts marks for new inputs.

6.-->Evaluation**
   - Model performance is checked using Mean Absolute Error.
